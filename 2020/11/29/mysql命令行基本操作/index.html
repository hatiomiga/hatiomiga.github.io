mysql命令行基本操作

基本操作

  • 创建数据库

    CREATE DATABASE IF NOT EXISTS `数据库名`

  • 删除数据库

    DROP DATABASE IF EXISTS `数据库名`

  • 使用数据库

    USE `数据库名`

  • 查看所有数据库

    SHOW DATABASES

  • 创建数据库表

  • 查看已存在数据库的语句

    SHOW CREATE DATABASE `表名`

-查看数据表的定义语句

SHOW CREATE TABLE `表名`

  • 查看表的具体结构

    DESC `表名`

  • 修改表名

    ALTER TABLE `旧表名` RENAME `新表名`

  • 增加表的字段

    ALTER TABLE `表名` ADD `字段名` 字段属性

  • 修改表的字段(1.修改属性，2.重命名)

    1.ALTER TABLE `表名` MODIFY `字段名` 字段属性
    2.ALTER TABLE `表名` CHANGE `旧字段名` `新字段名` 字段属性

  • 删除表的字段

    ALTER TABLE `表名` DROP `字段名`

  • 删除表

    DROP TABLE IF EXISTS `表名`
