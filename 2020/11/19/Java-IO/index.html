Java IO

java IO流详解

基本概念

1.流的输入和输出是相对内存的。
从内存出来为输出，
到内存去为输入。
(比如我从你的电脑拷贝文件这一行为，对于我的内存来说是输入
而对于你的内存则为输出）

2.流根据读取数据的方式分为字节流和字符流，字节流按照字节读取，
字符流按照字符读取，一次读取2个字节

3.字节流适合读取视频，声音，图片等二进制文件。字符流适合读取纯文本文件。

4.java语言中所有字节流都是以Stream结尾，
而所有的字符流都含有Reader或Writer。

##各种流的概括

  • 以File开头的与文件相关
  • 以Buffered开头的自带缓冲区
  • 以Object开头的专门读取Java对象
  • InputStreamReader和OutputStreamWriter为转换流(字节流转换成字符流)
